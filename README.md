# House Price Prediction

This project predicts house prices using Linear Regression.

Dataset: https://www.kaggle.com/datasets/harlfoxem/housesalesprediction

Steps:

1. Data preprocessing
2. Train test split
3. Model training using Linear Regression
4. Model evaluation using RMSE and R2 Score
   How to run:

pip install pandas scikit-learn matplotlib joblib python src/train.py python src/evaluate.py
